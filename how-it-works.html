How an LLM works?

From the moment you press Enter to the first word of the response,

here is what happens inside a large language model.

  1. 1

    User Input

    The user types a prompt into the interface.

  2. 2

    Tokenisation

    The text is split into smaller tokens.

  3. 3

    Embedding

    Tokens are converted into numerical vectors.

  4. 4

    Attention Layers

    The model focuses on important relationships between tokens.

  5. 5

    Prediction

    The model predicts the next token step by step.

  6. 6

    Output Generation

    The final response is generated and shown to the user.
